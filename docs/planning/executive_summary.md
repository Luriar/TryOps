> 면접 30초 / 본사 영업 첫 슬라이드 / CFO 결재 첫 페이지용 1pager.

---

## 한 줄 정의

**TryOps는 카메라가 불가능한 피팅룸 내부의 행동 신호(RuView/WiFi CSI)를 RFID·POS와 결합해, K-D2C 브랜드 본사가 *"고객이 입어봤지만 사지 않은 이유"* 를 추정하는 오프라인 리테일 전환 인텔리전스 SaaS다.**

---

## 핵심 명제 — 3단 구조

- **RFID는 *"무엇이 들어갔는지"*** 를 본다
- **POS는 *"결국 무엇이 팔렸는지"*** 를 본다
- **RuView/WiFi CSI는 그 사이의 *"망설임·반복 비교·체류 강도·대기 마찰·동행"*** 을 본다

→ TryOps는 RFID를 대체하지 않는다. **RFID의 가치를 확장**한다.

---

## 핵심 수치

| 항목 | 값 | 출처 |
|---|---|---|
| 1차 타겟 시장 | K-D2C 패션 본사 | - |
| SAM (좁은 정의) | 약 50~100 매장 | RFID 도입 한국 패션 5개 기업 × 평균 10~20매장 |
| SAM (넓은 정의) | 약 800~1,500 매장 | Flow Module 포함 K-D2C 본사 |

| Funnel Module 가격 | 매장당 월 15만원 (Pro) | 마진율 50% |
| Flow Module 가격 | 매장당 월 5만원 | 손실 사업, 시장 진입 도구 |
| 본사 ROI | 보수 5배 / 기본 15배 | 가정 민감도 분석 |
| 손익분기 (본사) | 1~5개월 | 도입 비용 vs 회수 |
| 영업 사이클 | 8~13개월 (대기업) / 6~9개월 (K-D2C) | 페르소나 시간 분해 |

## 결정적 차별점 4가지

### 1. 법적 우위 (🟢 검증됨)
**개인정보보호법 제25조** — 영상정보처리기기 규제. 피팅룸·탈의실에 카메라 설치 불가. WiFi CSI는 영상 촬영 아님. **TryOps만 합법적으로 피팅룸 내부 데이터 수집 가능**.

### 2. 시장 진입 GTM (🟡 추정)
한국 패션 RFID 도입 기업 약 5~10개로 좁음. **Flow Module로 K-D2C 본사(안다르·젝시믹스 등) 진입 → 본사 RFID 도입 → Funnel 업그레이드** 의 자연스러운 GTM 사이클.

### 3. 시점 정당성 (🟢 검증됨)
- 2024~2026: K-패션 글로벌 확장 변곡점 (안다르 미국·싱가포르·호주, 젝시믹스 일본·중국)
- 본사 데이터팀 보편화 (F&F 디지털본부, 코오롱FnC DT PU 실재)
- 패션 디지털 전환 압박 (KPMG 2026 보고서)

### 4. 4주 PoC + 검증 시스템 (🟡 추정)
- K-패션 시즌 사이클 2~3개월의 절반에 해당하는 4주 PoC
- 신뢰도 표기(🟢/🟡/🔴) + 회사 실재성 vs 페르소나 행동 분리 검증
- 페르소나 21건 1차 출처 + 셀프리뷰 v1 → v4 사이클

## 가장 큰 리스크 3가지

### 리스크 1: 손님 거부감 SNS 위기
오픈서베이 2024 *"오프라인 환경 개인정보 불안 76.2%"* (맥락 경고 적용). 매장 비영상 센서의 실제 거부감 비율은 미검증. 1건의 SNS 위기가 사업 흔들 가능.

**대응**: 매장 안내문 v2 + 직원 5개 표준 질문 매뉴얼 + 24시간 SNS 모니터링

### 리스크 2: 본사 결재 8~13개월 사이클
페르소나 분석 결과 영업 사이클 길음. 4주 PoC만으로 영업 진입 안 됨. 첫 본사 도달까지 자금 계획 필요.

**대응**: PoC vs Pilot 구분, 4주 PoC는 *"본사 내 확장 의사결정 도구"* 로 위치 명확화

### 리스크 3: RFID 솔루션사의 행동 분석 진입
셀메이트·idro·Nexite 등이 *"피팅룸 행동 분석 모듈"* 추가 출시 가능성.

**대응**: 첫 1~2년 안에 RFID 도입 5곳 중 최소 2~3곳 잠금 + CSI 기술 우위 (RuView 기반) + 솔루션사와 파트너십

## 한 줄 영업 메시지

**시나리오 A (Funnel Module, 대기업)**:
> *"이미 도입하신 RFID에 행동 신호를 더해 RFID ROI를 강화합니다. 본사 ROI 보수 5배 / 기본 15배, PoC 4주에 검증해드립니다."*

**시나리오 B (Flow Module, K-D2C)**:
> *"글로벌 매장 확장 시점에 매장 데이터 표준화의 도구입니다. 피팅룸 안의 행동 신호를 영상 없이 잡아드립니다. RFID 미도입 매장도 즉시 적용 가능."*
