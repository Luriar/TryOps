## 1. ROI 모델 — 본사 관점 (구매 의사결정자)

본사가 TryOps를 도입할 때 *"얼마가 들어가고 얼마가 돌아오는가"* 의 의사결정 모델.

### 1.1 비용 (Cost) — 매장당 월

| 항목 | 금액 (만원/매장/월) | 신뢰도 | 출처 |
|---|---|---|---|
| Funnel Pro 구독료 | 15 | 🟡 | TryOps 가격 정책 |
| 일회성 도입비 (감가 1/36) | 약 0.8 | 🟡 | 30만 / 36개월 |
| 본사 데이터팀 인력 비중 | 약 2~5 | 🔴 | 본사 분석 작업 인건비 분담 |
| 매장 운영 적응 비용 | 약 1~2 | 🔴 | 직원 교육·매뉴얼 변경 |
| **총 비용** | **약 19~23** | 🟡 | 매장당 월 |

### 1.2 효익 (Benefit) — 매장당 월

**효익 1: 전환율 개선**

```
추가 매출 = 월간 try-on 세션 수 × 개선 가능한 전환율 상승폭 × 평균 객단가 × 기여율
```

기본 가정 (각 가정의 신뢰도):

| 변수 | 값 | 신뢰도 | 출처 |
|---|---|---|---|
| 월간 try-on 세션 수 | 3,000건 | 🔴 가설 | 매장 일평균 100건 × 30일, PoC 측정 필수 |
| 현재 try-on purchase rate | 25% | 🔴 가설 | 패션 리테일 평균 추정 |
| 개선 가능한 상승폭 | +1.5%p | 🔴 가설 | RFID 도입 사례 기반 보수 추정 |
| 평균 객단가 | 70,000원 | 🟡 추정 | 애슬레저 평균 |
| 기여율 (TryOps 외 변수 보정) | 50% | 🟡 가정 | 보수 가정 |

계산:
```
3,000 × 0.015 × 70,000 × 0.5 = 1,575,000원 / 월 = 약 157만원/매장/월
```

**효익 2: 가격 인하 회피 (마진 방어)**

가격 문제 아닌 상품을 할인 안 함으로써 마진 방어.

| 변수 | 값 | 신뢰도 |
|---|---|---|
| 평균 가격 인하율 | 20% | 🟡 |
| 인하 회피 가능 SKU 비율 | 10~30% | 🔴 |
| 평균 SKU당 월 매출 | 50만원 | 🔴 |
| 매장당 SKU 수 | 100~200 | 🟡 |

계산 (10% 인하 회피 가정):
```
200 SKU × 0.10 × 50만 × 0.20 = 200만원/매장/월 마진 방어
```

🔴 본 계산은 가설 의존도 높음. PoC에서 정확한 인하 회피 비율 측정 필수.

**효익 3: 상품기획 ROI (장기)**

- 신상품 출시 초기 실패 원인 후보 발견
- 다음 시즌 사이즈 가이드·디자인 개선
- 반품·리뷰 발생 전 조기 신호

🔴 정량화 어려움. 본 ROI 모델에서는 *"보조 효익"* 으로 처리.

### 1.3 본사 ROI 종합 (매장당 월)

| 항목 | 금액 |
|---|---|
| 효익 1 (전환율 개선) | 약 157만원 |
| 효익 2 (마진 방어) | 약 200만원 |
| **총 효익** | **약 357만원** |
| 총 비용 | 약 19~23만원 |
| **순효익** | **약 334~338만원/매장/월** |
| **ROI 배수** | **약 15~18배** |

🔴 **중대 경고**: 위 ROI 15~18배는 모든 가정이 낙관적으로 맞을 때의 *"이상 시나리오"*. 실제는 훨씬 낮을 가능성 높음. 섹션 3 민감도 분석 참조.

## 2. 시나리오 분석 — 보수·기본·낙관

ROI는 가정 의존도 매우 높음. 3가지 시나리오로 분기.

### 2.1 보수 시나리오 (Conservative)

가정:
- 월간 try-on 세션 1,500건 (절반)
- 개선 상승폭 +0.5%p (1/3)
- 기여율 30% (보수)
- 인하 회피 SKU 5% (1/2)

| 항목 | 금액 |
|---|---|
| 효익 1 (전환율) | 약 16만원 |
| 효익 2 (마진 방어) | 약 100만원 |
| 총 효익 | 약 116만원 |
| 총 비용 | 약 23만원 |
| 순효익 | 약 93만원 |
| ROI 배수 | **약 5배** |

### 2.2 기본 시나리오 (Base)

가정: 섹션 1.2 기본값

| 항목 | 금액 |
|---|---|
| 순효익 | 약 334만원 |
| ROI 배수 | **약 15배** |

### 2.3 낙관 시나리오 (Optimistic)

가정 (보수 시나리오의 반대 방향, 단 비현실적 수준은 아님):
- 월간 try-on 세션 4,500건 (1.5배)
- 개선 상승폭 +2.0%p (1.33배)
- 기여율 60%
- 인하 회피 SKU 20%

| 항목 | 금액 |
|---|---|
| 효익 1 (전환율) | 약 378만원 |
| 효익 2 (마진 방어) | 약 400만원 |
| 총 효익 | 약 778만원 |
| 순효익 | 약 755만원 |
| ROI 배수 | **약 25~30배** |

### 2.4 시나리오 비교

| 시나리오 | ROI 배수 | 본사 의사결정 |
|---|---|---|
| 보수 (5배) | 충분히 매력적 | 도입 결재 가능 |
| 기본 (15배) | 매우 매력적 | 즉시 도입 |
| 낙관 (25~30배) | 매우 매력적 | 적극 도입 |

**영업 메시지로 사용할 ROI**: **5~15배** (보수~기본). 낙관 시나리오는 *"피팅룸 의존도 매우 높은 카테고리 + 본사 액션 매우 적극적"* 의 조건부 시나리오로 인용 가능.

## 3. 민감도 분석 — 어떤 변수가 가장 결정적인가

### 3.1 변수별 ROI 민감도

각 변수를 ±50% 변화 시 ROI 영향:

| 변수 | 기본값 | 변화 시 ROI 영향 | 민감도 |
|---|---|---|---|
| 월간 try-on 세션 수 | 3,000건 | ±50% → ROI ±50% | **★★★★★ 매우 높음** |
| 개선 가능한 상승폭 | 1.5%p | ±50% → ROI ±50% | **★★★★★ 매우 높음** |
| 평균 객단가 | 7만원 | ±50% → ROI ±50% | ★★★★ 높음 |
| 기여율 | 50% | ±50% → ROI ±50% | ★★★★ 높음 |
| 인하 회피 SKU 비율 | 10% | ±50% → ROI ±35% | ★★★ 중간 |
| 매장당 SKU 수 | 150 | ±50% → ROI ±35% | ★★★ 중간 |
| TryOps 가격 (15만원) | 15만원 | ±50% → ROI ±5% | ★ 낮음 |

**결정적 가정 2개**:
1. **월간 try-on 세션 수**: PoC 1순위 검증 (매장 입구·피팅룸 카운트 측정)
2. **개선 가능한 상승폭**: PoC 2~3개월 측정 후 검증

이 두 변수가 ROI의 본질. 가격 인상·인하는 ROI 영향 미미 (가격 정책 변경의 자유도 큼).

### 3.2 손익분기점 (Break-even) 분석

본사 입장: *"도입 비용 = 회수 매출"* 의 손익분기점.

| 기본값 대비 | 손익분기 도달 시점 |
|---|---|
| 보수 시나리오 | 약 5개월 |
| 기본 시나리오 | 약 1~2개월 |
| 낙관 시나리오 | 약 1개월 이내 |

**도입 결재 메시지**:
> *"보수 가정에서도 5개월 안에 회수됩니다. 기본 가정에서는 1~2개월. PoC 4주에 가설을 검증해드립니다."*

## 4. ROI 계산의 결정적 불확실성

### 4.1 가장 큰 가설 3개

본 ROI 모델이 깨질 수 있는 결정적 가설:

**가설 1: TryOps가 진짜로 전환율을 올린다 (인과관계)**

- 본 모델은 *"TryOps 도입 = 전환율 +1.5%p"* 의 인과관계 가정
- 실제는 *"분석 도구 도입 = 본사가 액션 안 함 = 효과 0"* 일 수 있음
- 실패 시나리오 4 (본사가 데이터 봐도 액션 안 함) 참조 (`product_strategy.md`)

**가설 2: 매장당 try-on 세션 수가 충분하다**

- 본 모델은 *"월 3,000건 = 일평균 100건"* 가정
- 실제 한국 매장 try-on 세션 평균 데이터 없음
- 매장 사이즈·위치·시간대에 따라 큰 편차 (10건~500건)
- 일평균 30건 이하 매장은 ROI 자동 1/3 미만

**가설 3: 본사 액션이 효과를 낸다**

- 본 모델은 *"본사가 분석 결과를 보고 가격 인하 결정을 바꾼다"* 가정
- 실제는 *"임원이 직관으로 그냥 인하한다"* 일 수 있음
- 페르소나 분석 결과 거부형 임원 (김상철) 영향력으로 액션 자체가 안 일어날 가능성

### 4.2 ROI 검증 우선순위

PoC 단계 검증 우선순위:

1. **Try-on 세션 수 실측** (Week 1, 모든 PoC 매장)
2. **Hesitation Score → conversion 상관관계** (Week 2~4)
3. **본사 액션 채택률** (Week 4 → Pilot Week 12)
4. **개선 가능 상승폭 실측** (Pilot Week 4~12)
5. **마진 방어 효과 실측** (Pilot Week 12~24)

### 5.2 3년 SOM ARR 추정

| 시점 | 매장 점유 | 본사 점유 | ARR |
|---|---|---|---|
| 1년차 | 30~50매장 | 3~5 본사 | 약 3~5억원 |

### 5.3 비용 구조 (TryOps 운영)

매장당 한계 비용 (`funnel_vs_flow.md` 섹션 4.2 참조): 7.5만원/매장/월.

본사당 ARPU 평균 매장당 12만원 이상 유지 시 사업성 성립.

3년 평균 매장 150개 가정:
- 매출: 약 12만 × 150매장 × 12개월 = **약 2.2억원/년**
- 한계 비용: 7.5만 × 150 × 12 = **약 1.4억원/년**
- 매장당 마진: 약 4.5만원/월 × 150 × 12 = **약 0.8억원/년**

🔴 위 계산은 *"평균 ARPU 12만원"* 가정 (현실은 Flow 5만 + Funnel Pro 15만 + Enterprise 10만의 혼합).

**TryOps 손익분기 임계점**: 본사 평균 ARPU 매장당 12만원 + 매장 100개 이상 운영.

## 6. 인터뷰가 흔들 수 있는 ROI 가정 (페르소나 v4 연계)

`personas_extended.md` v4 섹션 12.5.5 *"인터뷰가 바꿀 수 있는 사업 결정 Top 5"* 의 ROI 관점:

| 인터뷰 발견 | ROI 영향 |
|---|---|
| Flow vs Funnel 우선순위 변경 | TryOps 자체 ARR 추정 ±50% |
| 영업 사이클 12~18개월 (가정보다 길음) | 첫 본사 ARR 도달 시점 +6개월 → 자금 계획 변경 |
| 손님 거부감 비율 76.2% 실측 결과 | 매장 도입률 70% 가정이 50% 또는 30%로 하락 가능 |
| ARR 패키지 대신 분기 단위 결재 | 본사당 ARR 분할, 현금흐름 변경 |
| 거부형 페르소나 실제 영향력 | 영업 사이클 +3~6개월 추가 가능 |

이 5가지가 모두 **부정적으로** 흔들 시:
- 1년차 ARR: 3~5억원 → **1~2억원**
- 3년차 ARR: 10~20억원 → **5~10억원**
- 손익분기 도달: 3년차 → **5년차** 가능

긍정적으로 흔들 시 (예: K-D2C 본사 영업 가속, Flow 단독 도입 본사 다수 확보):
- 1년차 ARR: 5억 → **7~10억원**
- 3년차 ARR: 20억 → **30~40억원**

ROI 가정의 폭 → **사업 성패의 폭이 5~10배 차이**. PoC 검증의 중요성.

## 7.5 가격 협상 매뉴얼

본사 영업 시 가격 협상은 ROI를 정직하게 공유하면서 진행. 시나리오별 매뉴얼.

### 7.5.1 시나리오 A — 대기업 본사 (Funnel Enterprise ARR)

**1차 제안 가격**: 본사 ARR 2.4억원/년 (100매장 × 20만원/매장/월 × 12개월)

**협상 단계별 흐름**:

| 협상 단계 | 본사 반응 예상 | TryOps 대응 |
|---|---|---|
| 1차 제안 | *"비싸다, 매장당 단가가 너무 높다"* | ROI 보수 5배 시뮬레이션 제시 (월 157만원 vs 20만원) |
| 2차 협상 | *"ARR이 아니라 분기 단위 결재 가능?"* | 1년 약정 + 분기 청구 옵션 제공 (단가 +10%) |
| 3차 협상 | *"매장당 15만원이면 도입"* | 100매장 → 15만원 × 12개월 = 본사 ARR 1.8억원 수락. 단, 도입비 30만원/매장 별도 |
| 최종 | 합의 또는 결렬 | 결렬 시 PoC 4주 무료 제공 옵션 |

**최저 수용 가격**: 매장당 월 12만원 (한계비용 7.5만 + 마진 60%). 그 이하는 결렬.

### 7.5.2 시나리오 B — K-D2C 본사 (Flow Module 우선)

**1차 제안 가격**: 매장당 월 5만원 (Flow Standalone) + 도입비 30만원/매장

**협상 단계별 흐름**:

| 협상 단계 | 본사 반응 예상 | TryOps 대응 |
|---|---|---|
| 1차 제안 | *"기존 마케팅 컨설팅이 분석 다 해줘"* | TryOps는 *"매장 데이터"*, 마케팅 영역 무관 차별화 |
| 2차 협상 | *"PoC 4주 무료 가능?"* | PoC 4주 무료 + 매장 노드 무료 대여 (단, 영업 사이클 6~9개월 분산) |
| 3차 협상 | *"확장 시 가격은?"* | RFID 도입 시 Funnel 업그레이드 (매장당 15만원). 2년 약정 시 첫 6개월 무료 |
| 최종 | PoC 시작 또는 결렬 | 결렬 시 *"마케팅 컨설팅과 파트너십"* 경로 검토 (에코마케팅 등) |

**최저 수용 가격**: 매장당 월 3만원 (한계비용 5만원 미달 — 진입 도구로만 활용). Flow Standalone은 손실 사업이지만 SAM 확보·Funnel 업그레이드 경로 확보 목적.

### 7.5.3 가격 협상의 결정적 원칙

**원칙 1: ROI 정직 공유**
- 보수 5배 / 기본 15배 / 낙관 25~30배 모두 정직 공유
- *"낙관 시나리오 인용 금지"* 같은 자해 표현 피함 (낙관도 합리적 조건부 시나리오로 제시)

**원칙 2: 결재 단계별 가격 분해**
- 박정우(데이터팀장) 결재: 매장당 단가
- 김상철(본부장) 결재: 본사 ARR 일괄
- 임원회의 결재: 1년·3년 누적 ROI

**원칙 3: PoC 4주 무료 vs 유료**
- PoC 유료(매장 3개 × 4주 × 5만원 = 60만원): 본사가 진지함 입증
- PoC 무료: 영업 진입 도구로 사용, 단 본사 ARR 1.5억원+ 약정 시만

- CFO 결재용 핵심 자료

### 7.6.3 자료 3: 부서별 활용 매핑

### 7.6.4 자료 4: 4주 PoC 패키지

- ROI 정밀화 보고서
- Pilot 비용 약 1,000만원 (10매장 × 12주)
