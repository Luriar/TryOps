### 1.2 D2C 의류 매장 추정 (가설 + 추정)

🔴 한국 의류 소매업 전체 사업체 정확 수치는 KOSIS 직접 조회 필요. 본 단계에서는 다음 추정 방법론으로 접근:

**KOSIS 직접 조회의 한계 명시**: 통계청 국가데이터처 *"전국사업체조사"* 의 한국표준산업분류 47.41 의류 소매업 세부 사업체 수는 공개 데이터 검색만으로는 직접 추출 불가. KOSIS 사이트 직접 접속 + 통계표 다운로드 필요. 본 포트폴리오 단계에서는 추정 방법론으로 대체.

🟢 **참조 가능한 인접 산업 사업체 수 (2024)**:
- 한국 화장품 책임유통판매업체: 27,932개 (Cosmorning 2025, Statista 인용)
- 한국 스포츠 산업 사업체: 131,764개 (문체부 2024 스포츠산업조사)

→ 의류 소매업 사업체는 화장품 책임유통판매업체보다 많고 스포츠 산업 전체보다 적은 범위(약 5만~10만 추정)로 합리적 추정 가능.

🟡 **간접 추정 방법론**:
- 한국 전체 사업체 약 635만개 중 도매·소매업 비중 약 15~18% → 약 95만~115만개
- 그중 의류·신발 도소매업 약 10~12% → 약 9.5만~14만개
- 그중 직영점·플래그십·백화점입점 비중 약 25~30% → **약 2.4만~4.2만개**
- 우리 적용 카테고리(애슬레저·언더웨어·여성 컨템포러리) 약 20~25% → **약 4,800~10,500개**

🔴 **위 추정의 신뢰도 낮음**. PoC 단계에서 KOSIS의 한국표준산업분류 47.41 의류 소매업 세부 사업체 수 직접 확인 필요. 단, 추정 범위 자체는 *"우리 TAM 정의가 의미 있는 규모"* 라는 결론에 영향 없음.

🟢 **검증 자료 추가 (KOSIS 직접 조회 한계 부분 해소)**:

2026년 추가 검색에서 확보된 1차 출처:

- **2023년 전체 사업체 수 6,238,580개** (통계청 *"2023년 전국사업체조사 결과(잠정)"*, 2024.09)
- **2023년 도소매업 사업체 5만 3천 개 증가 (+3.5%)** (같은 출처)
- **2024년 한국 의류 제조 사업체 수: 23,507개** (한국패션산업협회 *"섬유패션산업활성화기반마련"* 사업 통계, 패션비즈 2026.02)
- **의류 제조 사업체 종사자 8만 9,337명, 서울 59.2% 집중** (같은 출처)
- **의류 제조 5인 미만 사업장이 81.3%** (같은 출처)

출처:
- 통계청 2023 전국사업체조사: https://kostat.go.kr/board.es?act=view&bid=233&list_no=432931&mid=a10301010000
- 패션비즈 의류 제조업 실태: https://fashionbiz.co.kr/article/224079

🟡 **의류 제조 23,507개 → 의류 소매 추정 정밀화**:
- 의류 산업의 통상 제조 vs 소매 비율 1:3~1:5
- → 의류 소매 사업체 추정 **약 70,000~115,000개** (제조 23,507 × 3~5)
- → 그중 브랜드 본사 운영 매장 약 20~25% → **약 14,000~28,750개**
- → 우리 적용 카테고리 약 20~25% → **약 2,800~7,200개**

본 추정은 기존 *"약 24,000~42,000개 매장 → 4,800~10,500개"* 보다 보수적. KOSIS 직접 조회 시 정밀화 가능.

🔴 한국 의류 소매업 전체 사업체 정확 수치는 여전히 KOSIS 직접 조회 필요 (검색만으로 정확 수치 추출 불가). 진짜 사업 시작 시 KOSIS 사이트 직접 접속 + 통계표 다운로드 필요.

## 2. 한국 패션 RFID 도입 현황 — 정밀 검증

### 2.1 v1의 핵심 가정 재검토

v1에서 인용한 *"국내 패션 RFID 도입 약 10개 안팎"* 출처는 fpost.co.kr의 단일 기사였다. 이 가정은 본 v2에서 추가 검증되었다.

### 2.2 RFID 도입 확인된 한국 패션 기업 (검증)

🟢 **삼성물산 패션부문 - 에잇세컨즈 (8seconds)**: RFID 도입 결정, 시스템 구축 후 추동 신상품 출고 시점부터 전 매장 적용. 물류~매장 모두 적용.

🟢 **코오롱인더스트리 FnC부문**: RFID 도입 결정, 시스템 도입 진행 중

🟢 **에프앤에프 (F&F)**: RFID 도입 결정, 빠르게 시스템 도입 진행 중

🟢 **크리스에프앤씨 (골프웨어)**: 정부 RFID 지원사업 후 자체적인 연구와 전문 업체 협업으로 적용 성공, **지금까지도 효과적 운용**

🟢 **한세엠케이**: RFID 도입 사례 보유 (한세엠케이 김문환 대표 인터뷰 기사 — *"인력 절감, 재고관리 정확도, 로스율 제로에 가깝다, 안했으면 어쩔 뻔"*)
(출처: 한국경제 2019.04, 패션포스트)

🔴 **과거 도입 후 롤백한 기업들**: 제일모직(현 삼성물산 패션부문), 베이직하우스(현 티비에이치글로벌), 신세계인터내셔널, 스쿨룩스 — 2012년 정부 RFID 지원사업 참여했으나 *"크리스에프앤씨를 제외한 나머지 브랜드는 모두 포기하고 시스템을 롤백"*
(출처: 패션포스트 *"RFID 누구나 할 수 있다"*)

### 2.3 RFID 도입 패션 기업 — 검증된 명단 (5곳)

| 기업/브랜드 | RFID 상태 | 매장 모델 | TryOps 영업 가능성 |
|---|---|---|---|
| 삼성물산 패션 (에잇세컨즈) | 도입 완료, 전 매장 | SPA 직영점 | ★★★★ (대기업이라 보수적이지만 RFID 활용 동기 강함) |
| 코오롱인더스트리 FnC | 도입 진행 | 다수 브랜드 | ★★★★ (코오롱스포츠 일본 진출과 연계) |
| 에프앤에프 (F&F) | 도입 진행 | 디스커버리·MLB 운영 | ★★★★★ (한국 패션 매출 1위, MLB 브랜드 글로벌화) |
| 크리스에프앤씨 | 도입 완료, 운영 중 | 골프웨어 | ★★★★★ (기존 RFID ROI 검증된 곳, 행동 분석 확장 매력적) |
| 한세엠케이 | 도입 완료 | NBA·TBJ 등 다수 | ★★★ (RFID 운영팀 노하우 보유) |

**v1 대비 변화**: v1은 *"안다르·젝시믹스·마뗑킴 등이 1차 타겟"* 이라고 했지만, **실제 RFID 도입 확인된 곳에 안다르·젝시믹스가 없다**. 이게 시장 분석의 결정적 발견.

### 2.4 시장 진입 전략의 재구성

위 발견으로 v3.1의 Beach-head 전략을 재검토해야 한다.

**원래 전략 (v3.1)**: 안다르 → 젝시믹스 → 마뗑킴 → 비비안 → 크리스에프앤씨

**수정된 전략 (v2 분석 결과)**:

**시나리오 A — RFID 도입 확인된 곳 우선 (Funnel Module 직판)**:
1. 크리스에프앤씨 (RFID 운영 성공 사례, 행동 분석 확장 명분 명확)
2. 에프앤에프 (한국 패션 매출 1위, MLB 브랜드 글로벌화 동반)
3. 삼성물산 패션 (에잇세컨즈) (대기업 영업 사이클 길지만 ROI 명확)
4. 코오롱FnC (RFID 도입 진행 중, 활용 도구로 동시 영업)
5. 한세엠케이 (RFID 운영팀 노하우 보유, 분석 확장 매력)

**시나리오 B — RFID 미도입 K-D2C 본사 (Flow Module 우선, Funnel 업그레이드 옵션)**:
1. 안다르 (매장 70여 개, 글로벌 확장 중, RFID 미도입 가능성 높음 — 검증 필요)
2. 젝시믹스 (글로벌 확장 중, 매출 2,741억)
3. 마뗑킴, 마르디 메크르디 (K-컨템포러리, 팝업·플래그십 중심)
4. 비비안 (남영비비안) (언더웨어, 피팅 의존도 높음)
5. 신세계인터내셔날 산하 신진 브랜드 (브랜드사 자체 영업, 본사 데이터팀 보유)

🔴 **검증 필요 사항**: 안다르·젝시믹스·마뗑킴의 실제 RFID 도입 여부. 검색 결과로는 미확인. 1차 영업 콘택트 단계에서 확인 가능.

### 2.5 시장 진입 전략 변화의 비즈니스 함의

이 발견이 TryOps 비즈니스 모델에 미치는 영향:

| 항목 | v3.1 가정 | v2 검증 후 수정 |
|---|---|---|
| Funnel Module 1차 영업 대상 | K-D2C 브랜드 (안다르 등) | **삼성물산·에프앤에프 등 RFID 도입 대기업** |
| Flow Module 위상 | SAM 확보 핵심 | **여전히 핵심 + K-D2C 본사 영업의 메인 진입 도구** |
| 첫 본사 영업 사이클 | 4~8주 | **대기업은 6개월 이상 가능성 (보수성·결재 사이클)** |
| Beach-head 메시지 | "RFID 도입의 행동 분석 확장" | (RFID 도입 곳) "RFID ROI 강화 도구" + (K-D2C) "RFID 도입 전 행동 분석 우선 도구" |

**결정적 시사점**: 한국 패션 RFID 도입 기업이 5~10개에 불과한 현실에서, **Funnel Module이 진짜 메인 수익원이 될 수 있는가** 의 근본 질문이 생긴다. SAM이 너무 좁다.

→ **Flow Module이 진짜 메인일 가능성**, Funnel은 *"Flow 도입 후 RFID 도입 시 업그레이드"* 의 보조 모듈로 재포지셔닝하는 시나리오도 검토 필요.

이건 product_strategy.md v3.1의 *"Funnel이 메인, Flow는 확장"* 명제 자체를 흔드는 발견이다. 본 시장 분석이 비즈니스 모델 재검토를 요구하는 수준.

## 6. Beach-head 영업 시나리오 — 정직한 재구성

### 6.1 v1 셀프리뷰 비판의 반영

v1은 *"안다르 1곳에서 4주 PoC"* 시나리오를 적었지만:
- 안다르 RFID 도입 미확인
- 안다르 의사결정자 콘택트 경로 불명
- *"Week 0 본사 콘택트"* 한 줄로 영업 사이클 시작을 단순화

v2에서는 정직하게 재구성한다.

### 6.2 시나리오 A — RFID 도입 대기업 1곳 (Funnel Module)

**대상**: 에프앤에프 또는 삼성물산 패션 (RFID 도입 확인됨)

| 단계 | 작업 | 현실적 사이클 |
|---|---|---|
| Pre-Sales 1 | 본사 IT/데이터팀 콘택트 (LinkedIn/추천/박람회) | 1~2개월 |
| Pre-Sales 2 | NDA 체결, 정보 교환 | 1~2개월 |
| Sales 1 | PoC 제안서 제출, 임원 보고 | 1~2개월 |
| Sales 2 | 법무·개인정보·구매팀 검토 | 2~3개월 |
| PoC 0 | PoC 매장 선정, 기술 통합 | 1개월 |
| PoC 1-4 | 4주 PoC 수행 | 1개월 |
| Post-PoC | 결과 보고, 확장 의사결정 | 1~2개월 |
| **합계** | | **8~13개월** |

**현실 인식**: 첫 RFID 도입 대기업 영업은 1년 사이클로 가야 한다. 4주 PoC는 사이클의 *"한 단계"* 일 뿐.

### 6.2 시나리오 B — K-D2C 본사 (Flow Module 우선)

**대상**: 안다르 또는 젝시믹스 (RFID 미도입 가능성 높음, 매장·매출 검증됨)

| 단계 | 작업 | 현실적 사이클 |
|---|---|---|
| Pre-Sales | 본사 데이터팀/MD 콘택트 | 1~2개월 |
| Sales | Flow Module PoC 제안 (CSI 단독) | 1개월 |
| 검토 | 임원·법무 검토 | 1~2개월 |
| PoC 0 | 매장 1~2개 선정, 노드 설치 | 1개월 |
| PoC 1-4 | 4주 PoC | 1개월 |
| Post-PoC | 결과 보고, 확장 또는 Funnel 업그레이드 논의 | 1~2개월 |
| **합계** | | **6~9개월** |

**현실 인식**: K-D2C 본사는 의사결정 사이클이 대기업보다 빠르다 (조직 단순). 그래서 **Flow Module이 진입 도구로 더 적합**. RFID 도입을 우리가 권유하면 Funnel 업그레이드로 자연 진행.

### 6.3 4주 PoC의 진짜 의미

위 사이클을 보면 **4주 PoC는 영업 사이클의 절반 미만**이다. v3.1 product_strategy.md에서 *"4주 PoC가 한 시즌의 절반"* 으로 정당화했지만, **4주는 영업 사이클 전체의 일부**다.

실제 4주 PoC가 의미 있는 시점:
- 첫 Beach-head 본사 1곳을 잠근 후 (영업 사이클 거의 끝나는 시점)
- 그 본사가 *"3개 매장에서 4주 검증"* 으로 확장 의사결정을 하는 단계
- **즉, PoC는 *"한 본사 안에서 확장 결정의 도구"*, 영업 진입 도구가 아님**

### 6.4 첫 본사 영업 메시지

각 시나리오별 메시지:

**Funnel Module (RFID 도입 대기업)**:
> "이미 도입하신 RFID의 ROI를 극대화하는 행동 분석 레이어입니다. RFID는 '무엇이 들어갔다'까지 봅니다. 우리는 '그 안에서 무슨 일이 있었나'를 봅니다. PoC 4주에 high try-on / low conversion SKU 5개 이상 발굴해드립니다."

**Flow Module (K-D2C 본사)**:
> "글로벌 매장 확장 시점에 매장 데이터 표준화의 도구입니다. 피팅룸 안의 행동 신호를 영상 없이 잡아드립니다. RFID 미도입 매장도 즉시 적용 가능합니다. 향후 RFID 도입 시 Funnel Module로 업그레이드 가능합니다."

## 7. 5년 확장 로드맵 — 시나리오 분기 포함

### 7.1 v1 로드맵의 약점

v1 셀프리뷰에서 짚었듯이 *"일직선 낙관 시나리오"*. v2에서 분기 조건을 명시한다.

### 7.2 Year 1-2: Beach-head 확보

**낙관 시나리오 (시나리오 1)**:
- Funnel Module: 에프앤에프 또는 삼성물산 패션 1곳 확보 (50~100매장)
- Flow Module: 안다르 또는 젝시믹스 1곳 확보 (50~70매장)

**기본 시나리오 (시나리오 2)**:
- Flow Module: K-D2C 본사 2~3곳 확보 (총 100매장)
- Funnel Module: PoC 진행 중, 미확정
- ARR: 약 3~5억원

**비관 시나리오 (시나리오 3)**:
- 모든 본사 PoC 거절 또는 검토 단계에서 중단
- 시뮬레이션 데이터 기반 데모만 보유
- ARR: 0원, 추가 fund-raising 필요

**시나리오 1 → 2 → 3 분기 조건**:
- Year 1 6개월차 본사 콘택트 성공률 (>50% / 20-50% / <20%)
- Year 1 12개월차 PoC 계약 체결 (1건 이상 / 0건 / 0건)

### 7.3 Year 2-3: 카테고리 확장

**선행 조건**: Year 1-2에 시나리오 1 또는 2 달성

**기본 경로**:
- 애슬레저·언더웨어 확장 (안다르·젝시믹스 외 비비안·인심내의)
- K-컨템포러리 진입 (마뗑킴·마르디 메크르디)
- 신세계·한섬 산하 신진 브랜드

**분기 조건**:
- Year 2에 본사 5곳 이상 확보 → Year 3 카테고리 확장 시작
- Year 2에 본사 3곳 이하 → Year 3는 기존 본사 매장 확장 집중

### 7.4 Year 3-4: 백화점·대기업 진입

**선행 조건**: K-D2C 본사 10곳 이상 + ARR 20억원+

- 신세계인터내셔날 자체 브랜드 (SI VILLAGE)

### 7.5 Year 4-5: 글로벌 확장

**경로 A — K-패션 동반 글로벌**:
- 안다르가 미국 진출하면 미국 매장에 노드 도입

## 8. 시장 리스크 — 보강된 4종

### 8.1 RFID 도입 정체 (실증된 리스크)

🟢 **2012년 정부 RFID 지원사업 종료 후 도입 정체** — 다수 기업 시도 후 롤백
(출처: 패션포스트)

🟢 **현재 도입 확인된 패션 기업 5개** — 매우 좁은 시장
(2.3절 참조)

**대응 강화**:
- Flow Module을 메인 진입 도구로 전환 검토
- Funnel Module은 *"RFID 운영 중인 5곳의 안정적 영업"* 으로 한정
- 본사가 우리 권유로 RFID 도입 → Funnel 업그레이드 사이클 설계

### 8.2 기존 RFID 솔루션사의 행동 분석 진입 (잠재 리스크)

🟡 셀메이트·idro·Nexite 등이 *"피팅룸 행동 분석 모듈"* 출시 가능성

**대응 강화**:
- 첫 1~2년 안에 RFID 도입 5곳 중 최소 2~3곳 잠금
- 센서 융합·EPC 조인 기술 우위 명확화 (자체 IP)
- 솔루션사와 데이터 통합 파트너십 (그들 RFID + 우리 센서 신호 + Ghost-read 데이터 품질 모듈)

### 8.3 본사 의사결정 보수성 (검증된 리스크)

🟢 **한국 본사는 RFID조차 도입 후 다수 롤백** — 신규 IT 도입 매우 보수적

**대응 강화**:
- 4주 PoC가 영업 진입 도구가 아니라 **본사 내 확장 의사결정 도구**임을 명확히
- 영업 사이클을 6~13개월로 설정 (단기 영업 안 함)
- ROI는 *"RFID 도입 ROI 회복"* 또는 *"매장 확장 의사결정 도구"* 로 프레이밍

### 8.4 손님 거부감 SNS 논란 (잠재 리스크)

🔴 WiFi CSI가 *"피팅룸 감시"* 로 오해되어 SNS 논란 → 매장 즉시 도입 철회

**대응 강화**:
- 도입 전 매장 안내문·고객 안내 강화
- 손님 페르소나 인터뷰 (Phase 2 단계)
- 도입 본사와 *"위기 대응 매뉴얼"* 공동 작성

### 8.5 글로벌 SaaS 진입 (장기 리스크)

🟢 RetailNext·Nexite·Aware 같은 글로벌 SaaS의 한국 진출 가능성

**대응**:

## 8.5 페르소나 v3·v4 검증과 시장 분석 통합 (v3.4 신규)

페르소나 검증 과정에서 확보된 1차 출처들이 시장 분석에 미치는 영향. v1·v2 시장 분석에 통합 안 됐던 결정적 자료 3건을 본 섹션에서 정리.

### 8.5.1 F&F 스노우플레이크 도입 — 시장 진입 시점 정당성 강화

🟢 *"F&F는 디지털 트랜스포메이션의 성공으로 시스템이 다양해지고 데이터 량이 폭증. 데이터웨어하우스의 성능 저하, 분석 니즈의 다양화 등 새로운 과제 등장 → 스노우플레이크 기반 통합 데이터 플랫폼 구축"*.

출처: Snowflake Korea F&F 케이스 스터디
링크: https://www.snowflake.com/ko/customers/all-customers/case-study/fandf/

**시장 분석 함의**:
- 한국 패션 대기업의 데이터 인프라가 *"폭증 단계"* 도달
- 외부 데이터·분석 도구 도입 수요가 검증된 사실
- TryOps의 시장 진입 시점 정당성 강화 — *"본사가 분석 도구를 적극 도입하는 시기"*
- Funnel Module 영업의 *"이미 도입한 RFID·DW의 ROI 강화"* 메시지의 본질적 근거

### 8.5.2 한국경제 백화점 VOC 사례 — 매장 단위 거부 리스크 정량화

🟢 *"백화점 VOC 3회 접수 시 매장이 본사와의 계약 연장에서 큰 불이익. 알바생도 본사 정책을 정확히 알고 활용해 매장 폐업 야기"*.

출처: 한국경제 2024.12.01
링크: https://www.hankyung.com/article/202411278443i

**시장 분석 함의**:
- 매장 단위 거부가 본사 영업과 별개로 사업을 막을 수 있는 리얼 리스크
- *"본사 OK → 매장 도입"* 의 자동 가정이 깨짐
- Beach-head 본사 5곳 영업 성공 후에도 **매장 단위 도입률 50~70%** 가 현실적 가정 (100% 아님)
- SOM 추정에 *"본사 도입률"* × *"매장 단위 도입률"* 의 이중 곱셈 필요
- SAM (넓은 정의 800~1,500매장)의 실제 도달 가능 비율이 *"본사 도달률 30% × 매장 도달률 70% = 21%"* 수준일 수 있음

### 8.5.3 안다르 에코마케팅 의존 — K-D2C 본사 영업 시나리오 보강

🟢 *"안다르는 D2C 전문업체 에코마케팅과 손을 잡았다. 박효영 에코마케팅 CMO를 안다르 공동 대표로 선임. 사실상 사업 운영 전반을 에코마케팅에 맡긴 셈"*.

출처: bizwatch 2021.04
링크: http://news.bizwatch.co.kr/article/consumer/2021/04/16/0039

**시장 분석 함의**:
- K-D2C 본사가 외부 마케팅 컨설팅에 의존하는 패턴 검증
- 시나리오 B 영업 시 *"안다르 직접 콘택트"* 외에도 *"에코마케팅 통한 간접 영업"* 경로 가능
- 영업 채널이 본사 직접만이 아니라 **마케팅 컨설팅 파트너십 경로**도 있음
- TryOps가 에코마케팅과 파트너십 체결 시 안다르·클럭 등 자체 브랜드 동시 영업 가능

### 8.5.4 SAM/SOM 정정 — 페르소나 발견 반영

페르소나 v4의 위 3건 발견을 SAM/SOM 추정에 반영:

| 구분 | v2 표기 | v3.4 정정 (페르소나 발견 반영) |
|---|---|---|
| SAM (좁은) | 50~100 매장 | 50~100 매장 (변화 없음) |
| SAM (넓은) | 800~1,500 매장 | 800~1,500 매장 (변화 없음) |
| **SOM (3년)** | 200~300 매장 | **약 100~200 매장** (매장 단위 도입률 70% 가정 적용) |
| **SOM ARR** | 약 20억~30억원 | **약 10억~20억원** (보수 정정) |

SOM이 감소한 이유: 매장 단위 거부 리스크의 정량화로 실제 도달 가능 매장이 본사 도입 매장의 70% 수준으로 보수적 가정.

### 8.5.5 시장 진입 전략 추가 옵션 — 마케팅 컨설팅 파트너십

페르소나 v4 발견으로 GTM 옵션 1개 추가:

**옵션 E (신규): 에코마케팅·아이디어키 등 D2C 마케팅 컨설팅과 파트너십**
- 컨설팅이 운영하는 D2C 브랜드 (클럭·안다르 등) 즉시 진입
- 컨설팅 영업 인력 활용 (TryOps 영업 비용 절감)
- 단점: 컨설팅 수수료 (매출의 10~20% 추정) 분배 필요

이 옵션이 본사 직접 영업(시나리오 B)보다 빠를 수 있음. PoC 1단계에서 옵션 B vs 옵션 E 비교 검증 필요.
