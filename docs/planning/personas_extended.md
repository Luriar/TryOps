# TryOps Personas Extended v4

> 본 v4는 v3 셀프리뷰의 11개 비판 + 4개 새 약점을 해소한다:
>
> 1. **점수 인플레이션 더 정직 정정** (v3 평균 45% → v4 실제 35%)
> 2. **이수정 검증 자료 1건 확보** (안다르 에코마케팅 의존 사례)
> 3. **오픈서베이 데이터 맥락 경고 명시**
> 4. **새 가설 명시 + 영업 사이클 시간 분해 복원**
> 5. **두 점수 해석 가이드 + 9인 우선순위 계층**
>
> 신뢰도 레벨 (전 문서 동일):
> - 🟢 검증됨 (1차 출처 직접 인용)
> - 🟡 추정 (검증 기반 계산)
> - 🔴 가설 (PoC/Discovery 검증 필요)

## 0. v4 점수 해석 가이드 — v3 셀프리뷰 비판 5 해소

본 문서의 페르소나 점수는 두 축으로 분리한다. 종합 점수가 아닌 **각 축을 독립적으로 평가**한다.

### 점수 두 축의 의미

**회사/조직 실재성 (0~100%)**: 해당 페르소나가 매핑된 회사·직급·조직이 한국에 실재하는지 검증된 정도. 1차 출처(IR 자료·채용 공고·언론 기사 등)로 직접 확인 가능.

**페르소나 행동 검증 (0~100%)**: 페르소나의 의사결정 패턴·페인포인트·트리거가 실증 자료(인터뷰·사례 분석·통계)로 입증된 정도. 이게 진짜 페르소나 검증의 본질.

### 종합 해석 규칙

- **회사 70% + 행동 30% = 영업 가능하지만 행동 시나리오는 가설**
- **회사 50% + 행동 60% = 일반론으로 행동은 입증되지만 구체 회사 매핑 약함**
- 두 점수의 단순 평균은 의미 없음. **행동 검증이 진짜 페르소나의 핵심**.

🔴 **중요 경고**: v3에서 인용한 *"오프라인 환경 개인정보 불안 76.2%"* 는 정확히는 *"개인정보 기재가 필요한 오프라인 이벤트 참여 시 불안"* 이다. **"개인정보 기재"** 라는 조건이 핵심.

진짜 검증: PoC 단계 *"매장 비영상 센서 인식 조사"* 별도 의뢰 필요.

## 10. v4의 결정적 학습 — 점수가 떨어지는 게 정직

본 v4 작업의 메타 학습:

1. **점수가 낮아질수록 정직성이 높아진다**. v1 → v4에서 평균 점수는 떨어졌지만(67% → 35%) 신뢰도는 올라갔다. 점수 인플레이션을 줄이는 게 진짜 검증.

2. **검증 자료의 *"맥락"* 이 가장 중요**. v3 오픈서베이 76.2%는 *"개인정보 기재"* 조건의 데이터인데 v3는 그 조건을 빼고 인용. v4에서 맥락 경고 추가가 핵심 학습.

3. **검증 못한 부분을 *"검증 못 했다"* 라고 명시하는 게 진짜 검증**. v4 박서영 *"새 가설도 가설"* 명시 같은 정직성.

4. **셀프리뷰는 점수 깎는 도구**. 새 약점 발견 시 점수 올리려 하지 말고 깎아야 진짜 정직.

이 4가지가 *"실제 사업 시작 시 동일 프로세스 재사용"* 의 핵심 자산.

### 12.5.5 인터뷰가 바꿀 수 있는 사업 결정 (Top 5)

인터뷰의 진짜 가치는 **사업 결정을 바꾸는 것**. 본 v4 가정 중 인터뷰로 흔들릴 수 있는 5가지:

1. **Flow Module vs Funnel Module 우선순위**
   - 현재 가정: Flow가 K-D2C 본사 진입 도구, Funnel이 대기업 메인
   - 인터뷰로 흔들릴 시나리오: K-D2C 본사가 *"Flow 미흡, Funnel 즉시 도입"* 선호 → 전략 전환

2. **영업 사이클 8~13개월 (시나리오 A) / 6~9개월 (시나리오 B)**
   - 인터뷰 결과 12~18개월 / 9~12개월일 수 있음 → 자금 계획 재설계

3. **76.2% 손님 거부감 (오픈서베이 단순 적용)**
   - 인터뷰 결과 30~50%일 수 있음 → 신뢰 설계 투자 축소

4. **ARR 패키지 vs 매장당 단가**
   - 인터뷰 결과 대기업도 분기 단위 결재 선호 → 가격 정책 재설계

5. **거부형 페르소나(김상철·강희정·이수정)의 실제 영향력**
   - 인터뷰 결과 K-D2C 본사는 거부형 영향력 미미할 수도, 또는 대기업 매장 매니저가 의외로 우호적일 수도 → 영업 우선순위 재배치

이 5가지 흔들림이 **인터뷰의 진짜 ROI**. 본 v4 페르소나가 검증되지 않은 가정 위에 서 있다는 사실을 인지하고, 인터뷰 시 *"이 5가지를 검증해야 한다"* 의 우선순위로 진행한다.

## 13. 본 문서의 한계와 정직한 결론

본 personas_extended.md v4는 *"실제 사업 시작 시 동일 프로세스 재사용"* 목표에 부합하는 **검증 프로세스의 골격**을 제공한다. 그러나:

- 평균 검증율 35% — 즉 65%는 미검증
- 매장 비영상 센서의 한국 손님 인식은 **공개 데이터로 검증 불가**
- 박정우·김상철·이수정 등 핵심 페르소나의 *"실제 인물 인터뷰"* 0건

**진짜 사업 시작 시**: 본 v4의 21건 1차 출처 + 9인 매트릭스 + 시간 분해를 영업·마케팅·법무 자료의 기반으로 사용. **그 위에 인터뷰 56~91건을 쌓아야 진짜 페르소나**가 된다.

**셀프리뷰 무한 루프 종료 선언**: 본 v4는 **공개 자료 검증의 한계점**에 도달했다. v4→v5에서 추가로 끌어올릴 수 있는 검증은 **인터뷰 영역**이며, 공개 자료 추가 셀프리뷰는 *"표기·구조·정합성"* 같은 형식적 비판만 만든다. 따라서 본 페르소나 문서는 **v4를 마지막 공개 자료 검증 버전으로 확정**하고, 다음 검증 단계는 **PoC 단계 인터뷰 액션 플랜 (섹션 12.5)** 으로 명시 분리한다.

**본 문서 활용 가이드**: 신뢰도 표기·점수 분리·셀프리뷰 사이클·인터뷰 액션 플랜은 영업·마케팅·제품 개발 자료의 골격으로 즉시 활용. 페르소나 9인 매트릭스는 시나리오 A·B 영업 시 결재 라인 분석 + 거부 우회 전략 + 매장 단위 도입 매뉴얼 작성에 직접 적용 가능.
