# TryOps 영업 데모 시나리오 (B2B 패션 본사 타겟)

## 1. 인트로: 오프라인 매장의 '블랙박스' 문제
- **문제 제기**: "온라인 쇼핑몰에서는 장바구니 이탈률, 체류 시간을 다 봅니다. 하지만 오프라인 매장은 어떤가요? 피팅룸에 옷을 3벌 들고 들어가서 1벌만 샀다면, 나머지 2벌은 왜 안 샀는지 알 수 있나요?"
- **TryOps의 솔루션**: 카메라 없이, 와이파이 전파(CSI) 교란만으로 피팅룸 안의 행동을 분석합니다. 개인정보 침해(카메라/음성)가 전혀 없기 때문에 법적 리스크 없이 즉시 전 매장에 도입 가능합니다.

## 2. 실시간 데모 (매장 매니저 뷰)
- **상황 1: Assistance Need (도움 필요)**
  - 고객이 5분째 옷을 갈아입지 않고 서성입니다 (사이즈 교환 필요 등).
  - 매니저 태블릿에 🟡 **알림 발생**: "3번 피팅룸 고객이 도움을 필요로 할 확률 85%"
  - 매니저가 사이즈를 교환해주며 응대 성공.
- **상황 2: Phantom Detection (도난 의심/오류)**
  - RFID 태그는 피팅룸 안에 5벌 있다고 나오는데, CSI 분석 결과 안에는 사람이 없습니다.
  - 🔴 **도난/태그 은닉 의심 알림** 발생.

## 3. 본사 대시보드 데모 (HQ 임원 뷰)
- **Fitting Friction (피팅 마찰도) 분석**
  - "이번 주 출시된 A 블라우스, 피팅 횟수는 500회인데 구매 전환율이 10% 밖에 안 됩니다."
  - "평균 피팅 시간이 타 블라우스 대비 2배(지퍼 불편 등 결함 의심)입니다."
  - 본사 기획팀은 즉시 A 블라우스의 패턴/부자재 결함을 수정할 수 있습니다.
- **Companion Effect (동행자 효과)**
  - 2인 이상이 피팅룸 커튼 앞에서 대화할 때 구매율이 30% 상승한다는 데이터를 보여줍니다.
  - 이를 기반으로 매장 레이아웃(피팅룸 대기석 추가 등) 개선 근거를 제시합니다.

## 4. 비용과 ROI (가장 중요한 포인트)
- 타사 비전(카메라) 기반 솔루션: 매장당 설치비 수백만 원, 매니지드 월 유지비 다수 발생.
- **TryOps**: ESP32-S3 기반 센서(원가 1만원대 이하) + 초경량 서버리스 클라우드(BigQuery) 아키텍처.
- **매장당 월 인프라 한계비용**: **약 950원**. 
- "전국 100개 매장에 도입해도 월 클라우드 비용은 극도로 저렴합니다. 구매 전환율을 단 1%만 올려도 투자금액을 즉시 회수할 수 있습니다."

## 5. 결론 및 다음 단계
- 파일럿 프로그램(Stage 1) 제안: "우선 주요 상권의 플래그십 스토어 1곳의 피팅룸 3칸에 단 반나절 만에 무공사로 설치해드리겠습니다. 2주 뒤 실측 데이터 리포트를 확인해보시죠."
